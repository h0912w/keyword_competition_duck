# QA 계획 v2.0

QA는 실제 CLI와 실제 DuckDuckGo 요청으로 수행한다. Mock 기반 DDG QA는 금지한다.

## QA 원칙 (절대 위배 금지)

**QA는 별도 QA 전용 소프트웨어가 아니라, 실제 사용자가 실행하는 동일한 파이프라인을 QA 에이전트가 직접 수행하는 방식으로 진행한다.**

| 원칙 | 설명 | 위반 예시 |
|---|---|---|
| **실제 파이프라인 사용** | `subprocess.run()` 등을 통해 실제 `kcpc.main` 모듈 실행 | `random.randint()`로 DDG 결과 시뮬레이션 |
| **실제 DDG 요청** | duckduckgo_search 라이브러리로 실제 검색 | Mock 함수로 가짜 결과 반환 |
| **실제 출력 검증** | 생성된 Excel/CSV 파일을 읽어서 검증 | 메모리상 데이터만 검증 |
| **동일 환경 사용** | 사용자와 동일한 설정, 입력, 출력 | QA 전용 별도 설정 사용 |

## 핵심 변경사항 (v2.0)

1. **Google API 제거**: API 설정 없이 DDG 자체 검증만 수행
2. **30가지 DDG 품질 검증 요소**: 다각도 분석으로 신뢰성 확보
3. **자동 테스트 단어 생성**: 빈도 기반 영어 단어 10개 자동 생성
4. **무한 반복 QA**: 예상값과 일치할 때까지 테스트 반복
5. **실제 파이프라인 강제**: `subprocess.run()`으로 실제 KCPC 실행 후 출력 검증

## 자동 테스트 단어 생성

### 빈도 기반 영어 단어 10개 생성

QA 실행 시 매번 **신규 영어 단어 10개**를 자동 생성합니다.

#### 빈도 분배

| 빈도 그룹 | 점수 범위 | 개수 | 예시 |
|-----------|------------|------|------|
| **고빈도** | 1-3 | 3개 | the, and, have |
| **중빈도** | 4-7 | 4개 | research, analysis, method |
| **저빈도** | 8-10 | 3개 | xyzqwerty, ultra-rare-term |

#### 생성 규칙

1. **고빈도(1-3)**: 영어 가장 빈도 높은 단어
2. **중빈도(4-7)**: 일반적인 기술/비즈니스 용어
3. **저빈도(8-10)**: 무작위 문자열 조합, 희귀 용어

#### 예상 결과 패턴

| 빈도 | 예상 DDG 결과 수 |
|------|------------------|
| 고빈도(1-3) | 8-10 (매우 높음) |
| 중빈도(4-7) | 4-7 (중간) |
| 저빈도(8-10) | 0-3 (매우 낮음) |

## 무한 반복 QA 로직

### QA 패스 기준

테스트 결과가 **예상 패턴과 일치**해야 PASS:

1. **고빈도 단어**: DDG 결과 수 ≥ 7
2. **중빈도 단어**: DDG 결과 수 3~7
3. **저빈도 단어**: DDG 결과 수 ≤ 3

### 무한 반복 조건

- **FAIL 발생 시**: 자동으로 QA 재실행
- **최대 재시도**: 제한 없음 (예상값과 일치할 때까지)
- **실패 로그**: 매 재시도마다 결과 기록

### 강제 중단

- 사용자가 `Ctrl+C` 입력 시 중단
- 중단 시 현재까지의 결과 리포트 작성

## 필수 검증 항목

### 기본 검증
- `--help` 옵션 동작
- txt/csv/xlsx 입력 실행
- 실제 DDG 요청 로그 확인
- DB 생성, 로그 생성
- 결과 파일 생성
- 컬럼/행 수 검증
- 실패 항목 계약 준수
- Ctrl+C/재개 검토

### DDG 품질 검증 (30가지)

#### P0 (필수)
1. **백엔드 교차 검증**: html vs lite 일치율 ≥ 80%
2. **재현성 검증**: 3회 반복 편차 ≤ 1

#### P1 (중요)
3. **시간대별 일관성**: 오전/오후 차이 ≤ 2
4. **지역별 변동**: kr-us-jp 차이 분석
5. **키워드 유형별 패턴**: 브랜드명 > 일반명사 > 롱테일 순으로 결과 수 높음

#### P2 (선택)
6-30. 나머지 25가지 요소

| 단어 | 빈도 | 예상 | 실제 | 일치 |

## 30가지 검증 결과

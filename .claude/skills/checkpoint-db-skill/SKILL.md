# checkpoint-db-skill

## 역할
SQLite 체크포인트 관리. 스키마 생성, 상태 복구, 1건 처리마다 commit한다.

## 공통 규칙
- `CLAUDE.md`의 금지사항과 입출력 계약을 우선한다.
- 출력 계약을 임의 변경하지 않는다.
- 실패 정보는 로그와 DB/QA 리포트에 남긴다.

# export-result-skill

## 역할
최종 결과 파일 생성. Results/Run_Summary/Failed_Items/QA_Summary 계약을 유지한다.

## 공통 규칙
- `CLAUDE.md`의 금지사항과 입출력 계약을 우선한다.
- 출력 계약을 임의 변경하지 않는다.
- 실패 정보는 로그와 DB/QA 리포트에 남긴다.

# file-io-skill

## 역할
입력 파일 로딩 및 출력 파일 저장 보조. txt/csv/xlsx 첫 번째 열만 읽고 원본 인덱스를 보존한다.

## 공통 규칙
- `CLAUDE.md`의 금지사항과 입출력 계약을 우선한다.
- 출력 계약을 임의 변경하지 않는다.
- 실패 정보는 로그와 DB/QA 리포트에 남긴다.

# ddg-measure-skill

## 역할
DuckDuckGo 검색 측정. intitle 쿼리, 최대 10개 결과, 딜레이와 재시도 규칙을 지킨다.

## 공통 규칙
- `CLAUDE.md`의 금지사항과 입출력 계약을 우선한다.
- 출력 계약을 임의 변경하지 않는다.
- 실패 정보는 로그와 DB/QA 리포트에 남긴다.

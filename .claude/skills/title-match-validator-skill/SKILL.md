# title-match-validator-skill

## 역할
검색 결과 title 필드에 원본 키워드가 대소문자 무시로 포함되는지 검사한다.

## 공통 규칙
- `CLAUDE.md`의 금지사항과 입출력 계약을 우선한다.
- 출력 계약을 임의 변경하지 않는다.
- 실패 정보는 로그와 DB/QA 리포트에 남긴다.
